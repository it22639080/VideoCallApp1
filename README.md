# Real-Time Chat & Video Calling Platform

A modern full-stack social communication web application built with **React**, **Node.js**, **Express**, **MongoDB**, **TanStack Query**, and **Tailwind CSS**. The system provides secure authentication, user onboarding, friend connections, real-time messaging, video calling, protected routes, and customizable UI themes.

---

## 🚀 Project Overview

This project is a real-time communication platform that allows users to create an account, complete onboarding, connect with friends, chat instantly, and make video calls. The application is designed with a modern UI, protected authentication flows, reusable custom hooks, and scalable full-stack architecture.

---

## ✨ Key Features

- 🔐 JWT-based user authentication
- 📝 Login and signup flows
- 📄 User onboarding flow
- 👥 Friends system
- 💬 Real-time chat functionality
- 📹 Video calling support
- 🎨 32 customizable UI themes
- 🚨 Protected routes
- 🛠️ Custom React hooks
- ⚡ Data fetching and caching with TanStack Query
